# Industrial Command Center

The dashboard has been rebuilt as a shadcn-based operations cockpit.

## Design Direction

- Industrial/utilitarian aesthetic.
- Dense but readable command-center layout.
- Light mode remains aligned to the Data Analyze Tool pale surface palette.
- Dark mode uses graphite surfaces with amber operational accents.

## Primary Views

- Overview KPIs.
- Protocol source health.
- Pipeline stages.
- Live event sample table.
- Test workflow commands.
- AI gateway and operator links.
