# Industrial Command Center

The dashboard is now a routed shadcn-based operations cockpit with a dedicated command center and separate domain views.

## Design Direction

- Industrial/utilitarian aesthetic.
- Dense but readable command-center layout.
- Light mode remains aligned to the Data Analyze Tool pale surface palette.
- Dark mode uses graphite surfaces with amber operational accents.

## Primary Views

- Command center overview.
- Pipeline view.
- Processing view.
- Historian view.
- Observability view.
- Integrations view.
