# Dashboard Design Direction

## Direction

Use an industrial control-room visual system: graphite surfaces, sharp borders, dense metrics, restrained status colors, and one signature signal-flow visualization.

## Screens

- Overview cockpit
- Stream explorer
- Processing and Flink view
- CDC view
- AI gateway view
- Ops/debug view

## Constraints

- Keyboard-accessible controls and visible focus states
- No color-only status communication
- Reduced-motion support
- Windowed live data to avoid rendering pressure
